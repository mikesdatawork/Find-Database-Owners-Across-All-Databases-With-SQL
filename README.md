## About-Process

<p>Here's some SQL logic to get the database owner across all databases ( not the specifically the db_owner right or schema dbo ), but the designated database owner name attached to the database. I've also taken the liberty of adding human readable time when the databases were created.</p>      


## SQL-Logic
```SQL
select
'database name' = db_name(database_id)
,   'owner name' = suser_name(owner_sid)
,   'database created'  = datename(dw, create_date) + ' ' + convert(char, create_date, 9) from
sys.databases
order by
db_name(database_id) asc

```
